小橙的幸运数

1. 题目概述

给一串a0 - a(m-1)，一个最终结果c，q次询问，每次给一个x，问x能不能在下面规则下变成c

规则：每次x只能变成x + a(x mod m)

2. 输入数据范围

1 ≤ m,q ≤ 2e5; 1 ≤ c ≤ 1e9; 0 ≤ ai ≤ 1e9; 1 ≤ xi ≤ 1e9

3. 关键思考

注意到x，c均可以是从0 - m-1的基础点变来的，所以从x到c必将先走树上路径，再走环，反复周期到达c.

所以可以倍增dp存储每一步的变化，然后查表;

也可以根据题目最后结果相同（均为c），建一个从c反向回找的图，跑反向bfs，看x能不能由c反变化而来

4. 如何实现

第一种：类似倍增lca，定义dp[i][j]表示从基础点j开始跑2^i步得到的增量，存储所有转移方式，然后贪心看x能不能加到c

第二种：先反向建基础点组成的图，找单周期步长cycle，然后把所有点都映射成基础点，从c%m跑bfs，看x%m和c%m的关系
